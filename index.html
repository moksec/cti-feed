A critical remote code execution (RCE) vulnerability (CVE-2026-1357, CVSS 9.8) exists in WPvivid Backup & Migration plugin versions up to 0.9.123. The flaw stems from improper error handling in RSA decryption (specifically, `openssl_private_decrypt()` failing and passing 'false' to the AES routine, creating a predictable encryption key) combined with a lack of path sanitization allowing directory traversal. This allows unauthenticated attackers to upload arbitrary malicious files (e.g., PHP files) for complete website takeover. Microsoft has patched a high-severity remote code execution (RCE) vulnerability, CVE-2026-20841, in Windows 11 Notepad versions 11.2510 and earlier. The flaw, described as an 'Improper neutralization of special elements used in a command ('command injection'),' allowed attackers to execute local or remote programs. Exploitation involved tricking users into Ctrl+clicking specially crafted Markdown links (e.g., `file://` for executables or `ms-appinstaller://` for applications) within a `.md` file opened in Notepad. The fix, delivered via the Microsoft Store, introduces warning prompts for non-HTTP/HTTPS URI schemes (e.g., `file:`, `ms-appinstaller:`, `mailto:`, `ms-settings:`, `ms-search:`) when clicked in Notepad.
