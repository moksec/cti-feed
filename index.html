Tirith is a new open-source, cross-platform tool that operates by hooking into user shells (zsh, bash, fish, PowerShell) to inspect and block command execution containing deceptive elements. It specifically addresses: Homograph attacks (Unicode lookalike characters, punycode, mixed scripts), Terminal injection (ANSI escapes, bidi overrides, zero-width characters), Pipe-to-shell patterns (e.g., `curl | bash`), Dotfile hijacking, Insecure transport (HTTP to shell, TLS disabled), Supply-chain risks (typosquatted git repos, untrusted Docker registries), and Credential exposure (userinfo URLs). The tool functions locally without network calls, command modification, or telemetry. It is supported on Windows, Linux, and macOS, with installation options including Homebrew, npm, apt/dnf, Cargo, Nix, Scoop, Chocolatey, and Docker. It does not provide protection for Windows Command Prompt (cmd.exe).
