Apple's 'Limit Precise Location' feature is available on iPhone Air, iPhone 16e, and iPad Pro (M5) Wi-Fi + Cellular models running iOS 26.3 or later. It restricts cellular networks to identifying a device's approximate location (e.g., neighborhood) rather than a precise street address. This setting does not affect emergency calls or location data shared with applications via Location Services or features like Find My.
